#🛠 clockin-bot 一鍵部署教學
1. 下載 clockin-bot
git clone https://github.com/govovo12/104-autologin.git
cd clockin-bot
2. 創建虛擬環境 venv
python -m venv venv
venv\Scripts\activate
3. 安裝必備套件
pip install -r requirements.txt
4. 第一次個人化設定
步驟	說明
修改 config.py	填入自己的 Telegram Bot Token / Chat ID / Outline路徑
執行 login_save_cookie.py	手動登入 104 保存 login_state.json
執行 record_positions.py	手動記錄 Outline連線按鈕位置
5. 測試
python scheduler_main.py
6. 注意事項
login_state.json 有效期大約30天，建議每月10號重新保存一次

打卡只在週一～週五執行

每次打卡或失敗，都會發Telegram通知
----------------------------------------------------------------------------
 新增啟動腳本（clockin_start.bat）

@echo off
start "" "C:\Program Files (x86)\Outline\Outline.exe"
timeout /t 8
cd /d C:\Users\user\Desktop\clockin-bot
call venv\Scripts\activate.bat
python scheduler_main.py
pause
進入「工作排程器」，新增一個任務。

指定動作為「啟動程式」，並指定 D:\clockin_start.bat。


每天早上定時啟動即可！
